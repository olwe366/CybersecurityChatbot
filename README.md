# 🤖 CyberSecurityBot (C# Console App)

## 📌 Overview
CyberSecurityBot is a simple console-based chatbot developed using the .NET Framework.  
The chatbot interacts with users, provides basic cybersecurity awareness, and demonstrates core programming concepts such as user input handling, conditional logic, file handling, and console UI enhancements.

---

## 🎯 Features
- 🔊 Voice greeting using a `.wav` file
- 🎨 ASCII art logo displayed at startup
- 👤 User name input and personalized responses
- 💬 Interactive chatbot conversation
- 🔐 Cybersecurity awareness responses (passwords, phishing, etc.)
- ⚠ Input validation for empty or invalid input
- ✨ Typing effect for better user experience
- 🌈 Colored console interface

---

## 🛠 Technologies Used
- C# (.NET Framework)
- Console Application
- System.Media (for audio playback)



## 📂 Project Structure
CyberSecurityBot/
